The SMART-IC Project Overview

Project Goal

Build a complete framework for smart monitoring and production optimization, reducing defect generation and waste across semiconductor manufacturing.

Main Pillars

AI-powered engine: distills relevant information from production and testing data.

Enhanced MES: uses that information for production-aware management and scheduling.
