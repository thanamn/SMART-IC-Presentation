Part III: Preliminary Study

Setting The Anomaly Threshold

Turning reconstruction error into a binary decision.

The Decision Logic

Anomaly detection is treated as a binary classification task. We compare the reconstruction error (RE) against an imposed threshold:

IF (RE > Threshold) THEN Anomaly

The threshold is set to guarantee a target False Alarm Rate (FAR) compatible with fab operations.

Why ROC & AUC?

Since the "perfect" threshold depends on business cost (Yield loss vs. Maintenance cost), we evaluate models using **ROC Curves**:

  • Threshold-less Measure: AUC (Area Under Curve) evaluates the model across ALL possible thresholds.
  • Top Performance: VAE/AE achieved **AUC = 0.9**, proving high discriminative power.
